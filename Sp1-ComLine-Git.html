The Command Line, Git and Github

What is the Command Line?

Command Line is where you use commands to navigate through your computer. Instead of using your mouse to open, close, save, delete, copy and etc, you type it in.
It is like if you had access to the kitchen of a restaurant instead of sitting and using the Menu and the waiters service.

6 examples of Commands and what they do:

  • pwd - print working directory (shows what directory you are in, all the way back to Home directory)
  • ls - list of files and folders inside directory
  • cd - change directory
  • cd .. - go up (back) one directory
  • mkdir - create a directory
  • touch - create a file

What is Git and Github

Git is a version control software which enables you to have multiple people working in the same project without overwritting each others inputs. Github is a cloud service to keep the Git projects remotely for easy access and security.
